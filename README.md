# Mentorship Node.js Starter

Quick starter for the mentorship Node.js project.

Setup

1. Install dependencies:

```bash
npm install
```

2. Install recommended dev dependency (optional):

```bash
npm install --save-dev nodemon
```

Run

```bash
npm run dev   # development (requires nodemon)
npm start     # production
```

Next steps

- Add dependencies like Express when needed.
- Add linting/formatting configs if desired.
